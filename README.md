## 🎯 About

Ostium OSINT Browser is a specialized web browser designed for digital investigation professionals, cybersecurity experts, and open-source intelligence (OSINT) analysts. Built on the open-source Chromium project and powered by Microsoft Edge WebView2, it provides a secure, feature-rich platform for your research.

### 🌟 Key Features

- **🔒 Complete Privacy**: No data collection, everything stays local
- **🛠️ Built-in Tools**: Over 50 OSINT-dedicated features
- **🚀 Portable**: No installation required, just extract and run
- **🔄 Always Up-to-Date**: Automatic updates via Edge WebView2
- **🎓 Accessible**: Designed for both professionals and beginners

## 🚀 Installation

### Prerequisites

- **Windows** (7, 8, 10, 11)
- **OpenJDK**: [Download](https://learn.microsoft.com/en-us/java/openjdk/download)
- **Edge WebView2**: [Download](https://developer.microsoft.com/en-us/microsoft-edge/webview2/consumer/?form=MA13LH)
- **Node.js 14+**: [Download](https://nodejs.org/)

### Quick Installation

1. Download the latest version from [Releases](https://github.com/icaza/Ostium-Osint-Browser/releases/)
2. Extract the archive
3. Launch `Ostium.exe`

**No complex installation required!**

## 🛠️ Technologies Used

Ostium OSINT Browser relies on proven open-source technologies:

- **[Chromium](https://www.chromium.org/)**: Open-source rendering engine
- **[Microsoft Edge WebView2](https://developer.microsoft.com/microsoft-edge/webview2/)**: Modern and secure runtime
- **[PlantUML](https://plantuml.com/)**: Diagram generation (LGPL)
- **[HtmlAgilityPack](https://html-agility-pack.net/)**: HTML parsing
- **[SQLite](https://www.sqlite.org/)**: Embedded database
